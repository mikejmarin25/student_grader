a) Project Description:
The Student Grader program allows users to input student names and their scores, calculates averages, and determines grades based on a grading scale. It demonstrates input/output, decision structures, loops, and functions.

b) Instructions:
1. Open the terminal and navigate to the project folder.
2. Run the program using the command: python student_grader.py
3. Use the menu to add students, view their grades, or exit.

c) Project Files:
- student_grader.py: The main program file.
- ReadMe_MaikolMarin.txt: Instructions and project details.

d) Python Packages:
This project uses only Python's built-in functionality and requires no additional packages.
